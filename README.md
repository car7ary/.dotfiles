# dotfiles

- [rcm](https://github.com/thoughtbot/rcm) is required.

## インストール

```bash
$ git clone https://github.com/car7ary/dotfiles ~/.dotfiles
$ cd ~/.dotfiles
$ ./init.sh
$ RCRC="./rcrc" rcup
```

## 管理

- rcup : リンク作成
- mkrc : RCM管理に追加
- rcdn : リンク削除
- lsrc : ファイル一覧

## ホスト毎のファイル

```bash
mkrc -o <file>
```
